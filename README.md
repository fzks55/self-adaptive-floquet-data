# self-adaptive-floquet-data
Numerical data for figures in the self-adaptive Floquet kicked-rotor manuscript.

## Organization

- `data/fig3/`: data for Fig. 3
- `data/fig4/`: data for Fig. 4
- `data/supp/`: data for supplementary figures

Each data file is named according to the corresponding figure panel, for example `fig3a.mat`, `fig3b.mat`, and `figS1a.mat`.

Only numerical data are provided. Simulation and plotting source codes are not included.
